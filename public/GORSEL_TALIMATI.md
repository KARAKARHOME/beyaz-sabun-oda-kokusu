# Görsel Dosyaları Ekleme Talimatı

Lütfen aşağıdaki görselleri `/Users/karakar/Desktop/karakarhome/beyazsabun/public/` klasörüne ekleyin:

## Gerekli Görseller:

1. **logo.webp** 
   - Logo dosyası
   - Header'da kullanılıyor
   - Önerilen boyut: 200x200px veya daha büyük (kare format)

2. **favicon.webp**
   - Favicon (site ikonu)
   - Tarayıcı sekmesinde görünüyor
   - Önerilen boyut: 32x32px veya 64x64px

3. **Beyaz Sabun Çubuklu Oda Kokusu.webp**
   - Ürün görseli
   - Ana sayfada hero bölümünde kullanılıyor
   - Önerilen boyut: 800x800px veya daha büyük

## Dosyaları Ekleme:

Bu görselleri `public` klasörüne kopyalayın:
```
/Users/karakar/Desktop/karakarhome/beyazsabun/public/
├── logo.webp
├── favicon.webp
└── Beyaz Sabun Çubuklu Oda Kokusu.webp
```

Görseller eklendikten sonra site otomatik olarak gösterecektir.

## Not:
- Tüm görseller .webp formatında olmalı (performans için)
- Dosya isimleri tam olarak yukarıdaki gibi olmalı (büyük/küçük harf duyarlı)
- Görseller eklenmeden site çalışır ama görsel yerleri boş görünür
