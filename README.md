# Beyaz Sabun Çubuklu Oda Kokusu - KARAKAR HOME

Premium çubuklu oda kokusu web sitesi. Next.js 14, TypeScript, Tailwind CSS ve Framer Motion ile geliştirilmiştir.

## Özellikler

- ✨ Modern ve benzersiz tasarım
- 📱 %100 Mobil uyumlu
- 🎨 Ferah renk paleti (koyu zemin yok)
- 🚀 SEO optimize edilmiş
- 🌍 GEO uyumlu (Pendik, İstanbul)
- ♿ Erişilebilir
- 🎭 Smooth animasyonlar
- 📊 Schema markup
- 🗺️ Sitemap ve robots.txt

## Sayfalar

- **Ana Sayfa**: Hero section, özellikler, kullanım alanları, SSS
- **Hakkımızda**: Marka hikayesi, vizyon, misyon, üretim süreci
- **Ürün Detayı**: Teknik özellikler, kullanım talimatları, tablolar
- **İletişim**: İletişim bilgileri, sosyal medya, hızlı destek

## Kurulum

```bash
npm install
npm run dev
```

Tarayıcıda `http://localhost:3000` adresini açın.

## Build

```bash
npm run build
npm start
```

## Teknolojiler

- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- Framer Motion
- Lucide Icons
